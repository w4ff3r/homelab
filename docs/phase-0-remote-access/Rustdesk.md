
## About

Rustdesk is a remote desktop tool that allows a device to remotely access another device on the same network. In tandem with Tailscale, this will allow the Homelab to be accessed while off the home network. 

## Setup:

1. Go to https://rustdesk.com/?utm_source=chatgpt.com
2. When clicking download, it will take you to a GitHub page
3. Download the version you need
