
## About

Rustdesk is a remote desktop tool that allows a device to remotely access another device on the same network. In tandem with Tailscale, this will allow the Homelab to be accessed while off the home network. 

## Setup:

1. Go to https://rustdesk.com
2. When clicking download, it will take you to a GitHub page
3. Download the version you need
4. Change settings to allow full access from security section (don't have to allow configuration changes but is useful)
5. Personally, I had to install again through the pink banner on the left of the window because the mouse was not syncing correctly in the remote desktop

![[Rustdesk-Downloaded.png]]

## How to use:

1. Once opened application shown above, enter the remote ID of the device you wish to connect to (found on left side of the window)
2. Input password when remote window opens
