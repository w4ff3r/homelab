
### About:

Tailscale acts as a secure private network where you use an email address to connect devices to one another. Once setup is complete, it doesn't need to be configured again (for now...)

### Setup:

1. Go to https://tailscale.com
2. Start creating an account
3. Once prompted, download Tailscale for the device you are starting on (In my case, the desktop where I will be running the Home Lab environment)
4. Open installer and follow steps
5. Click link to network (Use the email you used to register for the network link)
6. Follow above steps for another device (Important: Use same email address)
